ASteCA

The Automated Stellar Cluster Analysis package.

Package to automatically process and analyze star clusters. More information is available in the code's site.

View on GitHub





ezPADOVA-2

A python package that allows you to download PADOVA/PARSEC isochrones directly from the CMD website. Fork of the original ezpadova code by Morgan Fouesneau.

View on GitHub





UBV-move

Code to move stars in a UBV color-color diagram to their intrinsic position over a given ZAMS.

Uses an interpolated Schmidt-Kaler (1982) ZAMS and the standard extinction law:


E(U-B) = 0.72*E(B-V) + 0.05*E(B-V) ** 2

View on GitHub





arXiv

Attempts to imitate the myADS Notification Service.
Goes through the new submissions in one ore more categories in arXiv and presents those articles more relevant to the user, sorted according to a list of keywords selected by the user.

View on GitHub





massclean-cl

This set of bash/python scripts is intended to generate synthetic open clusters (SOCs) by making use of the MASSCLEAN package and to compare their real parameters (center, radius, metallicity, age, distance, extinction) with those obtained by the ASteCA code applied to each one of them.
