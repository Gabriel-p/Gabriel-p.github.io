IMF

Estimate the IMF for an observed cluster, given its individual masses (and uncertainties). The method is originally developed in Khalaj & Baumgardt (2013).

View on GitHub





Catalog Match

Given input ra, dec coordinates, uses the astroquery package to download the same frame from a selected catalog. Both catalogs are then matched using astropy's match_to_catalog_sky.

View on GitHub





astrometry

Takes an input data file with (x,y) coordinates and assigns (alpha, delta) astrometry, given a *_corr.fits file previously obtained from the astrometry.net service. The algorithm used to transform (x,y)-->(ra,dec) is the one developed by Jarno Elonen, based on the article Fitting affine and orthogonal transformations between two sets of points, H.
